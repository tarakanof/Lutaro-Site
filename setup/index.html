Set up your Sony camera for Lutaro

Menu paths verified against Sony's official α6700 Help Guide · July 2026

Lutaro talks to your camera the same way Sony's own remote software does: over Wi-Fi using the camera's PC Remote function, or (on Mac) over USB. This guide walks through every camera setting the app needs, with the exact menu paths.

Two menu generations. Newer Sony bodies (a9 III, α1 II, ZV-E1, and recent firmware on the a7 IV, a7R V, and α1) renamed the remote-control menus: Cnct./PC Remote → PC Remote Function became Cnct./Remote Sht. → Remote Shoot Function. The α6700 and most earlier bodies keep the classic names. Both are shown below; everything else is named the same on both generations.

1. Put the camera on your Wi-Fi network

Camera and device (iPhone, iPad, or Mac) must be on the same Wi-Fi network.

MENU → Network → Wi-Fi → Wi-Fi Connect = On

The camera searches for access points. If yours isn't listed, use WPS Push (press the WPS button on your router) or Access Point Set. (enter the network name and password manually) in the same Wi-Fi menu.

Recent firmware (α6700 2.00 and later) supports only WPA2/WPA3 networks — the camera won't join old WPA- or WEP-secured networks.

2. Turn on PC Remote

This is the switch that lets Lutaro connect at all.

BodiesMenu path
α6700 and most earlier bodies MENU → Network → Cnct./PC Remote → PC Remote Function → PC Remote = On
a9 III, α1 II, ZV-E1, updated a7 IV / a7R V / α1 MENU → Network → Cnct./Remote Sht. → Remote Shoot Function → Remote Shooting = On
The camera drops Wi-Fi when it sleeps. If Lutaro can't find it, wake the camera (half-press the shutter) and scan again.

3. Access Authentication — with or without a password

Sony cameras can protect remote connections with a user name and password. The setting is called Access Authen. and lives in the same place on both menu generations:

MENU → Network → Network Option → Access Authen. Settings → Access Authen.
  • On — the connection is authenticated and encrypted (SSH). Lutaro asks for the camera's user name and password. This is Sony's recommended mode.
  • Off — no password, no encryption; instead the camera asks you to approve each new device once (pairing, below). Sony's help guide warns the connection can be intercepted in this mode.

Where to find the user name and password

MENU → Network → Network Option → Access Authen. Info

This screen shows the camera's user name, password, MAC address, and fingerprint. The credentials are auto-generated at the factory; you can change them (or generate a fresh password) under Access Authen. Settings. Type the user name and password into Lutaro exactly as shown.

If you regenerate the password, saved connections in Lutaro will fail until you enter the new one — the app will tell you when that happens.

4. Pairing (only when Access Authen. is Off)

With Access Authen. Off, the first connection from a new device must be approved on the camera body:

BodiesMenu path
α6700 and most earlier bodies MENU → Network → Cnct./PC Remote → PC Remote Function → Pairing
a9 III, α1 II, ZV-E1, updated a7 IV / a7R V / α1 MENU → Network → Cnct./Remote Sht. → Remote Shoot Function → Pairing
  1. Open Pairing on the camera.
  2. Connect from Lutaro.
  3. Approve the prompt that appears on the camera screen.

The camera remembers up to 20 paired devices. If the list fills up (or after a camera initialization), clear it and re-pair:

MENU → Network → Network Option → Reset Network Set.
Reset Network Set. clears all network settings, including saved Wi-Fi networks — you'll need step 1 again afterwards.

5. USB (Mac only)

On the Mac, Lutaro can also connect over a USB cable. Set the camera's USB mode to remote control:

MENU → Setup → USB → USB Connection Mode → PC Remote

If USB Connection Mode is set to Sel. When Connect, the camera asks on every cable connect — choose Remote Shoot (PC Remote).

6. Troubleshooting

  • Camera not found on Wi-Fi — it's probably asleep; the camera drops Wi-Fi when idle. Wake it, confirm it's on the same network, and confirm PC Remote (step 2) is On.
  • "Camera is busy" — another app holds the connection. Close Imaging Edge Desktop, Creators' App, or any other Sony remote software, then retry. Sony cameras allow only one remote session at a time.
  • Wrong user name or password — re-check them on the camera under Network → Network Option → Access Authen. Info. They're case-sensitive.
  • "Fingerprint changed" — appears after a camera initialization or network reset, or if a different camera answers at the same address. If you reset the camera (or this is genuinely a different body), trust the new fingerprint; otherwise check your network.
  • Pairing prompt never appears — make sure the camera is sitting in the Pairing menu (step 4) before you connect from Lutaro.
